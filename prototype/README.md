# 느좋점수 (NeujotScore) — 프로토타입 (0단계)

PRD: [`../docs/PRD-느좋점수.md`](../docs/PRD-느좋점수.md)

설치·빌드 없이 **브라우저에서 바로 도는** 화면 프로토타입입니다.
핵심 흐름(업로드 → 평가 → 결과카드)과 **캔디 게이팅**, **목업 AI 즉석 추정**을 체험용으로 구현했습니다.

## 실행 방법

방법 1 — 그냥 파일 열기:
```
prototype/index.html 더블클릭 (브라우저로 열기)
```

방법 2 — 로컬 서버(이미지 fallback 등 안정적):
```bash
cd prototype
python3 -m http.server 8000
# 브라우저에서 http://localhost:8000
```

## 체험 시나리오
1. **내 사진 평가받기** → 사진 선택 → AI가 즉석 점수/무드 미리보기
2. **첫 사진은 무료** → "진짜 사람들의 평가 받기" → 결과 카드
3. 두 번째 사진부터는 🍬 부족 → **남 사진 10개 평가**(슬라이더+무드 태그)로 캔디 적립
4. 캔디가 모이면 자동으로 결과 카드 오픈 → **공유 이미지 저장**

## 구현/미구현
| 구현(목업) | 미구현(MVP에서) |
|---|---|
| 화면 3종, 화면 전환 | 실제 백엔드/DB(Supabase) |
| 목업 AI 점수·태그 | 진짜 비전 AI / NSFW 필터 |
| 캔디 게이팅(localStorage) | 로그인·계정 |
| 무드 슬라이더 + 긍정 태그 | 실제 타 유저 평가 수집 |
| canvas 공유 카드 PNG | 카톡/인스타 SDK 연동 |

## 설계 가드레일 (코드에 반영됨)
- 무드 태그는 **전부 긍정/중립** (`sample-data.js`의 `MOOD_TAGS`) — 부정 표현 없음
- 점수는 "사진의 무드"에 대한 것 (얼굴 우열 아님)
- 외모 **순위표 없음**, 자기 결과만 표시

## 참고: 평가용 샘플 이미지
실제 인물 사진은 권리·윤리 문제로 쓰지 않습니다.
무드 풍경/오브제(picsum.photos, 저작권 프리)를 쓰고, 오프라인이면 그라데이션으로 대체됩니다.
