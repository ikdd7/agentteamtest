# 🎀 백꾸핀 — 백꾸 덕후들의 핀터레스트 (MVP)

가방꾸미기(백꾸) **디스커버리** 서비스. 홈은 "쇼핑"이 아니라 **피드(구경)** 입니다.
다른 사람 백꾸를 둘러보다 → 쓰인 아이템을 발견 → **실제 판매처로 이동(아웃바운드)**.
재고·결제 없이 **트래픽 + 제휴 수수료**로 버는 오늘의집/핀터레스트형 모델.

## 바로 실행

설치·빌드 불필요, 외부 라이브러리 0.

```
prototype/bag-deco/index.html  → 브라우저로 열기
# 또는
cd prototype/bag-deco && python3 -m http.server 8000
```

## 핵심 화면

| 화면 | 설명 |
|------|------|
| 🏠 **피드(홈)** | 백꾸 핀들을 masonry로 구경. **검색 + 스타일 태그 필터**(인형·코퀘트·Y2K·프루티…) |
| 📌 **핀 상세** | 큰 미리보기 + **쓰인 아이템 목록**(브랜드·가격·사러가기) + ♡좋아요 · 🔖저장 · 따라꾸미기 |
| 🎨 **나도 꾸미기** | 가방+키링 배치 에디터. 완성 후 **피드에 올리기** |
| 🧾 **아이템 상세 카드** | 브랜드·가격대·재질·설명 (오늘의집 "이 집 아이템"형) |

## 트래픽 루프 (이 제품의 엔진)

```
피드 구경 → "이 키링 예쁘다" → 핀 상세 → 사러 가기(아웃바운드) ─┐
   ↑                                                          │
   └─ 따라 꾸미기 / 내 백꾸 올리기 / 저장·공유 ←───────────────┘
```

- **재방문 이유**가 생김(매번 새 백꾸) → 리텐션
- **콘텐츠가 콘텐츠를 부름**(올리기→구경→따라꾸미기→또 올리기)
- 구경 중 자연스러운 **아웃바운드 = 트래픽/수익** (하단에 이동 횟수 집계)

## 비즈니스 모델

- 재고·물류·결제 **직접 안 함** → 콜드스타트 없음
- KPI = **아웃바운드 클릭 · 저장 · 공유 수**
- 수익 = 제휴 수수료(데모: 네이버쇼핑 검색 링크) + 향후 작가 입점·스폰서 백꾸

## 지금은 데모라서 (다음 단계)

- 피드/저장이 **세션 메모리** → 서버(Supabase 등) 저장·실시간 피드 필요
- 제휴 **딥링크 연동**(쿠팡 파트너스·다이소몰·지그재그) → 정확 랜딩 + 수익화
- 인기순/최신순 정렬·랭킹, 크리에이터·팔로우, GA4 아웃바운드 트래킹
- 고가 참·디자이너 콜라보로 객단가 ↑

## 기술 메모

- 순수 HTML/CSS/Vanilla JS, 의존성 0
- 키링·가방은 자체 제작 **SVG 일러스트**(`assets/`) → 항상 정확·오프라인·저장 안정
- 실제 사진이 필요하면 가방/키링 **직접 업로드** 지원
