# 📡 빈자리 레이더

공공 **수영·테니스·유아체능·문화강좌**의 **취소표(마감 → 접수중 전환)를 감지해 알림**해주는 앱.
새벽 5시 광클·매크로 전쟁 없이, 자리가 나는 순간 바로 알려준다.

- **데이터**: 서울 열린데이터광장 공개 API (`ListPublicReservationSport/Education/Culture`) — 스크래핑이 아닌 합법 오픈API
- **의존성 0**: 파이썬 표준 라이브러리만 사용 (`pip install` 불필요)
- **1인 운영형**: 크롤/폴링 + 푸시 알림, 서버·프런트 단일 프로세스

## 실행

```bash
./run.sh
# 또는
python3 -m app.server
```

브라우저에서 **http://localhost:8000** 접속. 켜는 즉시 목록이 뜨고, 취소표가 발생하면
오른쪽 "🚨 취소표 알림"에 실시간으로 쌓인다. 별표(★)로 관심 등록한 항목은
"🔔 알림 켜기"를 누르면 브라우저 푸시 알림까지 받는다.

### 실시간 모드 (실제 서울 데이터)

```bash
SEOUL_API_KEY=발급받은키 ./run.sh
```

키는 https://data.seoul.go.kr 에서 무료 발급. 키가 없으면 자동으로 **데모 모드**로
동작하며, 샘플 데이터의 상태를 주기적으로 흔들어 취소표 발생을 시뮬레이션한다.

## 환경변수

| 변수 | 기본값 | 설명 |
|---|---|---|
| `SEOUL_API_KEY` | (없음) | 서울 공개 API 키. 없으면 데모 모드 |
| `RADAR_PORT` | `8000` | 서버 포트 |
| `RADAR_POLL_SEC` | 실시간 60 / 데모 20 | 폴링 주기(초) |

## 구조

```
app/
  server.py     # 표준 라이브러리 HTTP 서버 (정적 서빙 + API)
  poller.py     # 데이터 수집 + '접수중' 전환 감지
  config.py     # 설정
  data/demo.json# 데모용 샘플 데이터
  static/       # 프런트엔드 (index.html / app.js / styles.css)
```

## API

| 메서드 | 경로 | 설명 |
|---|---|---|
| GET | `/api/facilities?category=&area=&q=&watch=` | 시설·강습 목록(상태 포함) |
| GET | `/api/alerts` | 최근 취소표 알림 |
| GET/POST | `/api/watches` | 관심 목록 조회 / 등록 |
| DELETE | `/api/watches/{id}` | 관심 해제 |

## 로드맵

1. (현재) 서울 수영/체육/문화 통합 — 취소표 감지·알림
2. 카카오/웹푸시 연동, 관심 슬롯별 조건 알림(요일·시간대)
3. 경기·지방 지자체 예약 시스템 확장
4. 공식 대기열 API 제휴(B2G)로 크롤링 의존 탈피
