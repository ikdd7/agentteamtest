# 🎙️ 말로 하는 투두 (Voice To-Do)

음성으로 말하면 내용을 간추려 **날짜별·행동(카테고리)별 투두리스트**로 정리해 주는
나만의 음성 비서 웹앱입니다. 출퇴근길이나 여유로울 때 말로 일정을 등록하고,
완료 처리하고, 음성으로 오늘 할 일을 들을 수 있어요. 작은 목표는 화면 맨 위에
항상 떠 있어 볼 때마다 동기부여가 됩니다.

> **서버도 API 키도 필요 없습니다.** 브라우저 내장 음성 인식(Web Speech API)과
> `localStorage`만 사용해 완전히 내 기기 안에서 동작합니다.

---

## 바로 써보기

1. `voice-todo/index.html`을 크롬 / 엣지 / 사파리 **최신 버전**에서 엽니다.
   - 로컬에서 더블클릭으로 열어도 동작하지만, 마이크 권한 때문에 `http(s)`로 여는 걸 권장합니다.
   - 가장 쉬운 방법:
     ```bash
     cd voice-todo
     python3 -m http.server 8000
     # 브라우저에서 http://localhost:8000 접속
     ```
2. 마이크 버튼을 누르고 권한을 **허용**합니다.
3. 말해보세요!

> GitHub Pages로 배포하면 휴대폰에서도 바로 쓸 수 있습니다
> (Settings → Pages → 이 폴더 기준 배포). 음성 인식은 `https`에서 가장 안정적입니다.

## 이렇게 말하면 됩니다

| 하고 싶은 것 | 예시 음성 | 결과 |
|---|---|---|
| **할 일 추가** | "오늘 운동하기" | 오늘 / 운동 카테고리로 등록 |
| | "내일 오후 3시 회의 준비할 거야" | 내일 / 업무 / ⏰오후 3시 |
| | "모레 장보기 그리고 청소하기" | 한 번에 2개 등록 |
| | "금요일까지 보고서 제출" | 다음 금요일로 등록 |
| **완료 처리** | "운동 완료", "회의 준비 다 했어" | 가장 잘 맞는 할 일을 완료 |
| **스케줄 확인** | "오늘 할 일 뭐야", "내일 스케줄 알려줘" | 음성으로 읽어줌 🔊 |
| **목표 추가** | "목표 추가 매일 책 읽기" | 상단 목표 패널에 고정 |
| **목표 확인** | "내 목표 읽어줘" | 음성으로 읽어줌 |

마이크가 없거나 음성이 안 될 때는 화면의 **직접 입력** 칸에 같은 문장을 적어도 똑같이 동작합니다.

### 인식되는 표현들
- **날짜**: 오늘 / 내일(낼) / 모레 / 글피 / 이번 주 / 다음 주 / 월~일요일 / `N월 N일` / `N일` / `~까지`
- **시간**: 오전·오후·아침·점심·저녁·밤·새벽, `3시`, `오후 3시`, `3시 반`, `9시 30분`
- **행동 분류(자동)**: 운동 · 업무 · 공부 · 쇼핑 · 집안일 · 약속 · 건강 · 금융 · 연락 · 기타

## 기능
- 🎤 **음성 입력** — 한국어(`ko-KR`) 실시간 받아쓰기
- 🧠 **자동 정리** — 날짜·시간·카테고리를 추출해 분류
- 🗂️ **두 가지 보기** — 날짜별 / 행동별 그룹 전환
- ✅ **음성 완료 처리** — 말한 내용과 가장 잘 맞는 할 일을 찾아 완료
- 🔊 **음성 읽어주기(TTS)** — 조회·등록·완료 결과를 음성으로 (상단 🔊 버튼으로 끄기/켜기)
- 🌱 **목표 패널** — 작은 목표를 화면 상단에 항상 노출
- 💾 **로컬 저장** — 데이터는 이 브라우저에만 저장 (`내보내기`로 JSON 백업)
- 📱 모바일 친화적 반응형 UI

## 브라우저 지원 / 주의
- 음성 **인식**(SpeechRecognition)은 Chrome, Edge, 최신 Safari에서 지원됩니다.
  Firefox 등 미지원 브라우저에서는 직접 입력으로 사용하세요.
- 음성 **읽어주기**(SpeechSynthesis)는 대부분의 최신 브라우저에서 동작합니다.
- 마이크 권한이 필요하며, `https` 또는 `localhost`에서 가장 안정적으로 동작합니다.

## 개인정보
모든 데이터(할 일·목표)는 외부로 전송되지 않고 **사용자 브라우저의 localStorage**에만
저장됩니다. 음성 인식 처리 방식은 사용하는 브라우저의 정책을 따릅니다.
