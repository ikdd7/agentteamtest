# 앱 아이디어 로그 — 20대 사소한 관심사 · 니치 · 창의

> 30분마다 10개씩. 주제: 20대의 *사소한 관심사*에서 출발한 앱, 창의적으로 *관심을 끌* 것, *니치* 시장.
> 형식: 이름 — 컨셉 / 씨앗(사소한 관심사) / 왜 끌리나(밈성·욕구) / 경쟁 메모.
> ⚠️ 경쟁 메모는 지식 기반 빠른 표기(정밀 실측 아님). 유망한 건 따로 WebSearch로 검증.

## 배치 #1 (2026-06-16)

1. **백꾸 도감** — 가방에 단 키링·인형·스트랩을 자랑하고 *아이템 출처*를 태깅·공유. / 씨앗: 백꾸·인형 트렌드 폭발. / 끌림: 자랑+"그거 어디서 샀어?" 욕구. / 경쟁: 인스타 산발, 전용 앱 약함.
2. **다꾸 마켓** — 다이어리·폰꾸 꾸미기 공유 + 스티커/마테 중고 교환. / 씨앗: 다꾸 문화. / 끌림: 수집·교환·자랑. / 경쟁: 모트모트·인스타·번개(통합 약함).
3. **시향 일기** — 시향한 향수 기록·노트·"오늘의 향". / 씨앗: 향수 입문 붐. / 끌림: 취향 큐레이션. / 경쟁: Fragrantica·내향인앱(있음 — 차별 필요).
4. **기프티콘 금고** — 받은 기프티콘 유효기간 알림 + 안 쓸 것 거래. / 씨앗: 사장되는 기프티콘 다수. / 끌림: 실용+소액 이득. / 경쟁: 기프티스타(거래), 통합 관리는 빈틈.
5. **방꾸 룸투어** — 자취방 꾸미기 공유 + 아이템 출처 태깅(저예산·원룸 특화). / 씨앗: 자취 인테리어. / 끌림: 룸투어 관음+따라사기. / 경쟁: 오늘의집(저예산 자취 니치는 약함).
6. **플리 커버** — 상황·감정별 플레이리스트 + 표지 디자인 공유. / 씨앗: 플레이리스트 큐레이션. / 끌림: 감성 자기표현. / 경쟁: 스포티파이·유튜브(소셜 큐레이션은 빈틈).
7. **랜덤 데일리 미션** — 매일 사소한 도전(낯선 카페·새 길·낯선 메뉴) 인증·공유. / 씨앗: 무료함·소확행. / 끌림: 인증·도파민. / 경쟁: 약함(니치).
8. **별점 인생** — 영화 말고 *모든 경험*(맛집·여행·물건·순간)에 별점+한줄 기록. / 씨앗: 기록·평가 욕구. / 끌림: Letterboxd式 취향 아카이브의 일반화. / 경쟁: 분야별 산발.
9. **꿈일기 + AI 해몽** — 자고 일어나 꿈 기록 → 해몽·패턴 분석. / 씨앗: 꿈·해몽 검색수요. / 끌림: 미스터리+자기탐구. / 경쟁: 약함(니치).
10. **무물방** — 특정 정체성·관심사별 *익명 질문*방(인스타 질문스티커 전용화). / 씨앗: 익명 호기심. / 끌림: 익명 솔직함. / 경쟁: 에타·인스타(주제 특화는 빈틈).
