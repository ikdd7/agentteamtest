# 딥 디벨롭 워크스페이스 (니치 고민 → 발칙한 한국형 form)

> 철학: **빈 시장을 찾지 않는다. 니치한 인간 "고민"을 찾아 발칙한 form으로 푼다.**
> 레퍼런스 DNA:
> - **거지맵** = "초가성비로 배 채우고 싶다(근데 거지같아 보이긴 싫다)" 고민 → 거지 정체성을 *자조적 자랑*으로 뒤집은 form.
> - **엑셀주식창** = "회사에서 몰래 주식 보고 싶다" 고민 → *업무 화면 위장*이라는 발칙한 form. (데이터가 아니라 form이 천재.)
> - **대다모** = "탈모를 혼자 앓기 외롭다·수치스럽다" 고민 → 익명 안전지대 + 평생 여정.
> 공통: 작은 고민 + 그 고민의 *감정*(수치·눈치·죄책감·자랑·불안)을 정확히 건드리는 **발칙한 형식**.

## 디벨롭 루프 규칙 (30분마다)
- **하나의 FOCUS 아이디어를 매 틱 한 단계씩 깊게** 디벨롭한다(새 아이디어 남발 금지).
- 디벨롭 단계(순서대로, 막히면 반복·심화):
  1. 고민 정의(누가/언제/어떤 감정) 2. 발칙한 form(왜 이 형식이 천재인가)
  3. 바이럴 훅(왜 공유되나) 4. 차별 웨지(기존 대비 왜 이김) 5. 콜드스타트(1인 시딩)
  6. 수익/그릇 7. 네이밍·카피 8. MVP 스펙 9. 리스크·규제 10. 재점수
- 한 아이디어가 *충분히 익으면*(또는 죽으면) IDEA-LOG.md로 결론 옮기고, 고민뱅크에서 다음 FOCUS 선택.

## 니치 고민 뱅크 (한국형, 발칙한 form 후보)
- 회사에서 몰래 딴짓(주식·이직·쇼핑·웹툰·사이드플젝) — 들킬까 불안 + 죄책감 + 동질감
- 축의금/경조사 얼마 내야 하나 — 눈치·체면·돈, 아무한테도 못 물음
- 단톡방 답장·읽씹·이모티콘 눈치 — 사회적 피로
- 남친/배우자 몰래 비상금·소비 — 관계 금기
- 부모님께 말 못하는 진로·연애·빚 — 효 문화 압박
- 혼자 밥/혼자 영화 들키기 싫음(혹은 인증하고 싶음) — 혼족 정체성
- 다이어트·금연 작심삼일 들키기 싫음 — 수치 + 재시작
- 회사 점심 메뉴/회식 자리 눈치 — 매일의 미시 스트레스
- 모임에서 "내가 호구인가" 더치페이·계산 눈치
- 부모·자식 손절(절연) 죄책감 — 금기(단 경쟁 포화 확인됨)

# FOCUS #1 — "딴짓의 알리바이": 회사에서 몰래 딴짓하는 직장인의 위장 도구+익명 부족

## iteration 1 — 고민 정의 + 발칙한 컨셉 (2026-06-14)

**고민(누가/언제/감정):** 거의 모든 직장인이 근무 중 몰래 딴짓을 한다(주식 확인·이직 알아보기·쇼핑·웹툰·사이드프로젝트). 감정 3종: ①상사/동료에게 *들킬까 불안* ②혼자 하는 *죄책감·고립* ③"나만 이러나? 다들 이러지?"라는 *동질감 갈망*. 엑셀주식창이 ①만 풀었다면, ②③까지 푸는 게 빈 자리.

**발칙한 form(가설):** 겉은 "업무 생산성 대시보드/스프레드시트"처럼 위장되지만, 안에서 *딴짓을 관리*하고(예: 이직 지원현황·관심종목·위시리스트), 익명으로 "오늘의 딴짓"을 *같은 처지 직장인끼리* 공유하는 위장 커뮤니티. 엑셀주식창의 *일반화 + 커뮤니티화*.

**왜 지금/한국:** 한국 직장 눈치·감시 문화 + 부업/이직 보편화 + "조용한 사직(quiet quitting)" 정서. 딴짓은 죄가 아니라 *생존*이라는 세대 정서가 form을 정당화.

**초기 점수(가설, 검증 전):** 공급=욕구 14(들킴불안+동질감은 강력) / 수요 13 / 콜드스타트 12 / 경쟁 약함 ?(엑셀주식창류 위장툴·블라인드 인접 — *iteration 4에서 실측 필요*) / 그릇 ?(B2C 위장툴 수익 불명 — iteration 6) / 발칙함 15(만점급) — **잠정, 경쟁·그릇 미검증.**

**다음 iteration 숙제:** (2) 위장 form을 *구체적으로* 어디까지(가짜 업무 UI의 디테일·단축키 패닉버튼) / (3) 왜 공유되나(딴짓 자랑의 쾌감) / (4) 블라인드·엑셀주식창과 뭐가 다른가 실측.

## iteration 2 — 위장 form 실측 + 웨지 재조준 (2026-06-14)

**경쟁 실측(검색):** "위장/보스키" form은 **이미 상품화됨** — Boss Key, Boss Sensor(AI 얼굴인식 화면 숨김), Panic Button(크롬), Magic Boss Key 등 다수. 즉 *발칙한 위장 자체는 novel하지 않다*(엑셀주식창이 떴던 건 form이 새로워서가 아니라 *밈으로 공유*돼서). → **발칙함 점수 15→9로 하향**(form은 흔함).

**그래서 핵심 깨달음:** 위장 도구는 해자가 아니다(table-stakes). FOCUS #1의 진짜 빈자리는 보스키 유틸들이 *건드리지 않는* **②딴짓의 죄책감 완화 + ③"다들 이러지?" 동질감** = 감정/커뮤니티 층이다.

**그런데 그 커뮤니티 층은 블라인드와 충돌:** 익명 직장인 커뮤니티는 블라인드가 점령. "딴짓"만 떼어낸 좁은 각으로 차별화해야 하는데 — 그 차별점이 *얇다*. (블라인드에 "딴짓 자랑" 글 쓰면 끝.)

**디벨롭 판정:** FOCUS #1은 두 겹(위장=흔함, 커뮤니티=블라인드와 충돌)에서 약해짐. 잠정 점수 **~74**. 단, *완전히 죽이기 전에* iteration 3에서 한 번 더 좁혀본다 —
- 후보 좁힘 A: "딴짓 일반"이 아니라 **단 하나의 딴짓**(예: *회사에서 몰래 이직 준비*)에 집중한 위장 OS. 이직은 죄책감·들킴불안이 가장 크고 *지갑이 깊다*(채용·이직 시장) → 그릇·웨지 둘 다 개선 가능성.
- 후보 좁힘 B: form을 위장이 아니라 **"딴짓 인증·자조 챌린지"**(거지맵식 자조적 자랑)로.
**iteration 3 숙제:** 좁힘 A(몰래 이직 위장 OS)의 경쟁·그릇 실측 → 살아나면 추격, 아니면 고민뱅크에서 FOCUS #2로 교체.

## iteration 3 — 좁힘 A 실측 + 큰 재조준 (2026-06-14)

**경쟁 실측(검색):** 좁힘 A(몰래 이직)도 점령됨 — LinkedIn 스텔스("Open to Work, 리크루터만"), **리멤버**(패시브 후보 영입), **Switch**("커리어판 틴더", 위장·축약 프로필), Teal·Huntr(지원 트래커). 위장·매칭·트래커 전 구간에 플레이어 존재.

**FOCUS #1 판정: 은퇴(~72).** 위장=흔함, 이직매칭=리멤버/링크드인/Switch, 트래커=Teal/Huntr, 커뮤니티=블라인드. 강한 웨지 없음.

**🔑 그런데 — 디벨롭이 더 큰 걸 가르쳐줬다(재조준):**
> 나는 3 iteration 내내 FOCUS #1을 **"경쟁 있음"**으로 죽였다. 그런데 **거지맵·엑셀주식창도 경쟁이 있었다**(배달앱·맛집앱·온갖 증권앱·보스키앱). 걔넨 *빈 시장이라서* 뜬 게 아니라 **① 문화적 공명(한국 직장인 정서를 정확히) + ② 강제 공유성(밈처럼 안 퍼질 수 없음)** 때문에 떴다.
> → **이 철학에선 "경쟁 약함"은 부차적이다. 결정축은 [문화 공명] × [공유 강제성(밈성)] × [고민의 감정 강도].** 내 100점 루브릭(경쟁약함 15점)은 이 게임엔 과대평가다.

**디벨롭 규칙 업데이트:** FOCUS 판정 기준을 *경쟁 약함*에서 **"밈성(자랑·웃김·공감으로 안 퍼질 수 없는가) + 한국 문화 공명 + 감정 강도"**로 전환. 경쟁은 "약한가"가 아니라 "더 공명하는 form으로 이기는가"로 본다.

**FOCUS #2 지명(다음 iteration):** *축의금/경조사 눈치* 고민 — 한국 초특화, 아무한테도 못 묻는 돈+체면, **"축의금 계산기"는 즉시 공유되는 밈 소재**(카톡으로 "야 이거 봐"), 안전한 금기. iteration 4에서 발칙한 form + 밈성부터 설계(경쟁 제거가 아니라).

# FOCUS #2 — "호구 방지기": 축의금/경조사 적정가 판정 + 익명 또래 데이터 + 경조사 장부

## iteration 4 — 발칙한 form + 밈성 + 공명 설계 (2026-06-14)

**고민(감정 해부):** 한국인은 경조사마다 4중 불안 — ① *호구/민폐 공포*("너무 적으면 욕, 많으면 부담") ② *품앗이 계산*("걔가 내 결혼식에 얼마 냈더라? 그만큼 돌려줘야") ③ *체면*("아무한테도 못 물어봄") ④ *또래 비교*("남들은 이 관계에 얼마 내?"). 글로벌 앱은 이 정서를 절대 모른다 — 부조/품앗이는 한국 고유.

**발칙한 form(2층 구조):**
- **훅(바이럴) = "축의금 호구지수" 판정기:** 관계(직장동료/친구/사촌…)·친밀도·참석여부·식대·내 나이만 넣으면 → *또래 익명 데이터* 기반 "적정 구간 + [호구 / 적정 / 통 큰] 판정 + 한 줄 드립". 결과를 **공유 카드**로 뽑아 카톡·인스타로 "나 이 정도면 적정이래ㅋㅋ" 공유.
- **앵커(리텐션) = 경조사 장부:** 준 돈/받은 돈 기록(이건 경쟁 앱들 영역) + **"이 사람에게 받은 만큼 돌려줄 차례" 알림**(품앗이 자동 정산). 장부는 흔하니 *판정·또래데이터·정산알림*으로 차별.

**왜 안 퍼질 수 없나(밈성):** "호구 테스트"는 한국에서 검증된 바이럴 포맷(MBTI·테스트류) + **결혼 성수기(봄·가을)마다 재유행** + 결과가 *자조·자랑 둘 다*("나 호구였네ㅠ" / "적정이래"). 친구끼리 "너 얼마 냈어?"를 *대신 물어주는* 도구라 대화에 자연 삽입.

**문화 공명 15 / 감정 강도 14 / 밈성·공유 14** (한국 부조 문화 초특화, 글로벌 무경쟁).

**경쟁(부차):** 장부 앱(경조사노트·치부책 등) 다수 = table-stakes. *적정가 판정 + 익명 또래 데이터 + 밈 공유 카드*는 **빔**. → 웨지 = "장부 앱이 아니라, *불안을 푸는 판정·비교*가 메인".

**콜드스타트(영리):** 또래 데이터는 give-to-get 함정처럼 보이나 — 사람들이 *판정받고 싶어서* 자기 케이스를 자연 입력 → 그게 곧 데이터. 시드는 공개 통계(아임인 등 "식대별 축의금 기준" 콘텐츠)로 초기 판정 기준 부트스트랩. **데이터 0에서도 판정 가능**(공개 기준) → 콜드스타트 12→14.

**그릇:** 모바일 부조 송금 흐름(카카오페이式 축의금 송금)에 수수료/제휴 + 화환·답례품·상조 커머스 + 결혼 버티컬 광고. 부조는 연 수조원 유통 → 핀테크/커머스로 깊은 지갑.

**디벨롭 점수(밈성 가중) 잠정 ~86.** 관건: ① 판정기의 밈성이 *실제로* 터지나(테스트 포맷이라 유리) ② 장부 앵커로 리텐션이 잡히나 ③ 송금/커머스로 그릇이 차나.

**iteration 5 숙제:** 판정기 결과 카드의 *카피·드립* 톤 설계(공유 욕구 극대화) + "호구지수" 점수 산식(또래 분포 퍼센타일) 구체화 + 첫 화면 플로우.

## iteration 5 — 호구지수 산식 + 결과 카드 카피 + 첫 화면 플로우 (2026-06-15)

**호구지수 산식(또래 퍼센타일):** 입력(관계·친밀도·참석여부·식대·내 나이/연차)이 같은 또래 코호트의 금액 분포에서 *내 금액의 퍼센타일*을 낸다.
- 하위 <10%ile → **"짠물 🥶"**(민폐 위험) / 25~75%ile → **"적정 ✅"**(무난) / 상위 >90%ile → **"호구 🚨"**(통 큼=손해).
- 데이터 0일 땐 공개 기준(식대×관계 계수)으로 *가상 분포* 생성 → 실데이터 쌓이면 보정.

**결과 카드 카피(공유 욕구 = 자조+팩폭+귀여움):**
- 🚨 `호구지수 87 — 직장 동료한테 15만원? 다음 달 점심은 걔가 사야 수지 맞음.`
- ✅ `적정지수 52 — 욕도 칭찬도 안 듣는 완벽한 무난함. 한국인의 정석.`
- 🥶 `짠물지수 하위 8% — 청첩장 받고 이 금액이면... 관계 재고(再考)각.`
- 카드 하단 고정: `나도 호구지수 판정받기 →`(강제 노출 = 바이럴 루프).

**첫 화면 플로우(가입 0, 3초 판정 → 엑셀주식창식 즉시 가치):**
1. 랜딩: "청첩장 받았어? 얼마 낼지 **3초 만에** 판정" — 가입 없이 바로 시작.
2. 관계 칩 탭: 직장동료/친구/대학동기/사촌/상사/지인…
3. 슬라이더: 친밀도(1~5) · 참석여부(참석/불참/식권만).
4. 식대 토글: 일반홀 / 호텔.
5. → **판정 카드**(적정구간 ₩ + 호구지수 + 드립) + [카톡 공유] [내 장부에 기록].
6. 기록 → 장부 진입(리텐션) + "이 사람한테 받을 차례 되면 알려줄게"(품앗이 락인).

**왜 강력한가(요약):** 가입 없이 *즉시 가치* + *즉시 공유*(밈) + *기록으로 락인*(리텐션) — Calendly·엑셀주식창식 단순함. 판정(획득) → 장부(유지) → 송금/커머스(수익)로 깔끔히 흐름.

**디벨롭 점수 유지 ~86~88.** 강점 명료(밈성·공명·플로우). 

**iteration 6 숙제(다음 약점 공략):** ① 리텐션의 진실 — 경조사는 *드문 이벤트*(월 0~2회)라 장부만으론 재방문이 약할 수 있다(낚시·축의금 공통 약점). 재방문 트리거를 더 설계(친구 결혼 알림? 부조 송금 허브?). ② 그릇 산수 구체화(송금 수수료 vs 커머스 vs 광고 중 무엇이 메인인가) — $20M 경로를 숫자로.

## iteration 6 — 리텐션·그릇 정면돌파 (정직한 하향) (2026-06-15)

**① 리텐션의 진실(약점 인정):** 경조사는 본질적으로 *드문 이벤트*(대다수 월 0~2회). 장부 보강(품앗이 net-position 대시보드, 친구 경조사 알림=연락처 연동·프라이버시 주의)으로도 *데일리 앱*은 안 된다.
→ **정직한 재포지션:** 이건 데일리 네트워크 앱이 아니라 **"시즌 바이럴 획득 + 이벤트당 커머스 + 평생 장부(기록 보관소)"** 모델이다. DAU는 낮아도 된다 — 획득이 *판정기 바이럴*이라 CAC가 거의 0이고, 수익이 *이벤트당*이라 거래량으로 큰다(세무·웨딩 앱과 같은 결).

**② 그릇 산수(추정, 수치 불확실 명시):**
- 한국 연 결혼 ~19만·장례 ~37만 + 돌/기타 → 경조사 이벤트 수백만/년, 각 이벤트에 화환·답례품 수반.
- 메인 수익 = **화환·답례품 커머스**(affiliate/마진). 화환 객단가 ~8~15만, 건당 마진 ~2만 가정 → ARR 40억(~$3M)엔 **연 ~20만 화환** 필요. 한국 화환 총량(추정 수백만 건) 대비 *일부 점유*면 가능. (단 화환 시장 규모 공개수치 미확보 = 불확실.)
- 보조: 결혼 버티컬 광고(웨딩홀·스드메), 비대면 부조 송금 제휴. **송금 수수료는 약함**(카페이·토스 무료와 경쟁).
- $20M 밸류 ≈ ARR $2~4M → 커머스 거래량으로 *도달 가능하나 ops가 무겁다*.

**정직한 텐션(중요):** 이 모델은 **네트워크 복리가 아니라 커머스 매출**로 $20M에 간다. 즉 네가 처음 말한 *"네트워크 효과로 300억"* 테제와는 결이 다르다(이건 발칙한 바이럴 유틸 + 커머스). 또 커머스 운영(화환 공급·배송 CS)은 *순수 1인 바이브코딩 범위를 넘는다* → 1인 빌드 점수 하향.

**재점수(정직): ~82.** 강점: 밈성·문화공명·콜드스타트·발칙함(여전히 최상). 약점: 데일리 리텐션 부재 + 커머스 ops 부담 + 네트워크효과 아님.

**판정:** FOCUS #2는 *"한국형 발칙 바이럴 유틸"로는 A급(만들면 분명 화제)*, 하지만 *"네트워크 효과 $20M"* 기준엔 커머스 의존이라 **B+급**. → 보류하지 말고 iteration 7에서 갈림길: (A) 이대로 "바이럴 유틸+커머스"로 받아들이고 MVP로 / (B) 같은 *축의금 불안* 고민에서 *네트워크 복리가 되는* 다른 form 재발상(예: 부조 데이터가 쌓일수록 모두에게 가치 커지는 구조) / (C) 고민뱅크에서 FOCUS #3로. **사용자 판단 필요 지점.**

## iteration 7 — (B) 네트워크 복리 form 재발상: 해자를 "데이터"로 (2026-06-15)

**전환:** 해자를 *판정 기능*(복제 쉬움)에서 **"한국 부조 실거래 데이터"**(쌓일수록 방어)로 옮긴다. = **부조판 Levels.fyi.**

**왜 이전 '실거래가 풀' 비판을 피하나(중요):** 그땐 give-to-get *뇌물* 구조 + 검증불가 자기보고가 문제였다. 부조는 다르다 — ① 공급이 *판정받고 싶은 욕구*로 굴러감(뇌물 아님) ② *통계 집계*라 개별 노이즈 허용(적정 '구간'만 맞으면 됨) ③ 금액 단위가 작고 거짓 유인 약함. → 자기보고의 치명상이 완화됨.

**확장(데이터 자산의 수직 성장):** 축의금 → 부의금 → 돌·환갑·집들이 → **"한국 관계 비용 데이터"** 전체로. 품앗이 net-position까지 합쳐지면 *개인 관계 회계* 플랫폼.

**그릇 재구성:** (a) 화환·답례품 커머스(여전) + (b) **데이터 권위 기반 B2B**(웨딩·상조 업계 벤치마크·트렌드 리포트) + (c) 비대면 부조 송금 허브. 데이터가 방어선이라 *커머스 단독*보다 밸류 배수 유리.

**남는 약점(정직):** 리텐션은 여전히 이벤트성(데이터·네트워크는 *획득·방어*를 강화하지 그것이 *데일리 인게이지*를 만들진 않음 — Levels/Glassdoor도 그러함). 단 이건 치명상 아님(그들도 큰 가치).

**재점수: ~86.** 네트워크효과·방어 회복(데이터 해자) + 밈성·공명 유지. 리텐션만 이벤트성으로 잔존.

**판정: FOCUS #2 = "부조 데이터 네트워크 + 호구지수 바이럴 훅"으로 ripe에 근접(~86).** 이게 세션 통틀어 *발칙한 한국형 × 네트워크 복리* 둘 다 만족하는 첫 후보다.
**iteration 8 숙제:** MVP 스펙(판정기 + 익명 금액 수집 + 분포 시각화 + 장부 + 공유카드)을 1인 빌드 범위로 확정 → 충분히 익으면 IDEA-LOG에 "선두 후보"로 승격.

## iteration 8 — MVP 스펙 (1인 바이브코딩 범위) + 승격 (2026-06-15)

**최종 디벨롭 점수: ~86 (확정).** 강점: 밈성·문화공명·콜드스타트·데이터해자·1인빌드. 약점(잔존·치명아님): 이벤트성 리텐션 + $20M은 *커머스+데이터 B2B* 경로(순수 네트워크 DAU 아님).

# FOCUS #3 — 황혼육아(손주 보는 조부모) 커뮤니티 + 번들 전략 검증

## iteration 1 — 고민·진입 form + "번들(황혼육아+갱년기+노년건강)" 질문 정면돌파 (2026-06-16)

**고민(매일 터지는 훅 — 희귀 강점):** 손주를 전담/반전담하는 50~70대 조부모("할마·할빠"). 매일 새 질문(이유식·어린이집 갈등·며느리/사위와 돈·허리무릎·떼쓰기 대응) + 외로움 + 손주 자랑 + 하소연. → 커뮤니티 3대 연료(정보·자랑·하소연)가 *매일* 발생 = 대부분 후보(이벤트성)의 리텐션 약점을 정면 해결.

**진입 form:** ① 손주 자랑 피드(최저 진입장벽 데일리 훅) ② 익명 며느리/사위 갈등 토로방 ③ **"육아 시세"**(조부모 돌봄비/용돈 또래 비교 — 호구지수와 같은 메커닉 재활용). 카톡 "할마방"으로 시딩.

### 🔑 번들 질문 결론: "처음부터 묶으면 죽고, 나이 따라 확장하면 산다"
사용자 질문 = 황혼육아+갱년기+노년건강을 엮은 커뮤니티 성공 가능성. 답:
- **처음부터 broad 번들 = 낮음(~20%).** 이유 ①경쟁: broad "5060 여성 종합"은 **오뉴·시놀·네이버 밴드가 이미 점령**(실측) → 빈 칸 소멸. ②정체성 충돌: 갱년기(45~55, 아직 '중년'·종종 직장인)는 '할머니 육아'·'노년건강'과 자기규정이 달라 서로 밀어냄(아무도 "노인방"에 들어가기 싫어함). ③ narrow passion 희석: 대다모=탈모 only, 거지맵=가성비 only로 떴듯, 묶으면 "이건 정확히 나를 위한 것" 신호가 사라짐.
- **narrow 진입 → 나이 따라 확장 = 높음.** 같은 *한국 5060 여성*이 황혼육아(60±) → 갱년기는 오히려 더 이름(45~55) → 노년건강(65+)으로 **생애주기상 자연 확장**. 즉 번들은 *출시 전략*이 아니라 *3~5년 비전*으로 옳다. 진입은 가장 빈·날카로운 쐐기 하나로.
- **어느 쐐기?** 황혼육아 — narrow 빔(맘카페=2030 엄마 전용, 당근 모임만 산발), **매일 훅**, 카톡 시딩, 무규제. (갱년기=핑크다이어리류 인접 점유, 노년건강=당뇨와건강 등 점유 → 진입 쐐기로 부적합. 둘은 *확장* 단계로.)

**경쟁 실측:** broad 시니어=오뉴/시놀/밴드(점령). 황혼육아 narrow=공개 통합 부재(✅). 갱년기/노년건강=부분 점유(확장 시 정면승부 필요).

**콜드스타트:** 카톡 할마방(지역×영아/유아) + 손주 자랑 스레드 + 유튜브 쇼츠(며느리 갈등·떼쓰기 꿀팁)로 검색·공유 유입. 시니어는 앱설치보다 카톡 잔류율 압도적.

**그릇:** 매개결제 커머스(손주 선물·완구·유아식 — 결정은 조부모, 결제는 자녀카드 → 객단가↑) + 조부모 육아 클래스 + 관절/치아/실손 보험 리드.

**약점:** ①결제 주체 분리(돈=자녀, 사용=조부모) ②타깃 디지털 약자(카톡 의존) ③$20M은 커머스/리드 경로(네트워크 복리 아님).

**점수(밈성/공명·집착·경쟁공백·콜드스타트·그릇 가중): ~80.** 강점: *유일하게 매일 훅* + 진짜 빔 + 카톡 시딩 + 무규제. → FOCUS #2(호구방지기 86)와 함께 톱2.
