# 백꾸 — 서비스화 종합 기획서 (v1.0)

> 10개 직군 에이전트(기획·UX·디자인·아키텍처·AI·수익·GTM·타깃·법무·레드팀)가
> WebSearch 실측을 근거로 수십 라운드 검토해 합친 **"바로 서비스 가능"** 수준의 통합 기획서.
> 한 줄: **"내 가방을 본 사람이 '그 키링 어디 거예요?' 묻게 만드는 곳 — 발견부터 입양(구매)까지, 백꾸."**
> 모델: 오늘의집(사진 위 태그) × LTK(샵더룩) 의 **백꾸 버전**.

---

## 0. 가장 먼저 읽어야 할 3줄 (정직한 결론)

1. **수요는 실재한다.** 백꾸는 2023년부터 이어진 지속 트렌드, 라부부 KREAM 거래액 전년比 폭증, "저 키링 어디 거?"라는 미해결 질문이 매일 쏟아진다. **전용 플랫폼(발견+커뮤니티+커머스 통합)은 비어 있다.**
2. **하지만 두 개의 치명적 리스크가 있다.** ① **콜드스타트**(크리에이터가 인스타를 떠날 이유가 없음) ② **수익 구조**(어필리에이트 단독은 사업이 안 됨 — 키링 건당 수수료 300~900원). 레드팀은 "빌드 전에 30일 인스타 큐레이션 계정으로 검증하라"고 강하게 권고했다.
3. **그래서 전략은 "빌드 먼저"가 아니라 "검증 먼저, 얇게 빌드"다.** 아래 0→1 플레이북은 *앱이 아니라 콘텐츠 계정*부터 시작한다. 수익은 어필리에이트가 아니라 **입점·광고·자체 IP 굿즈**로 설계한다.

## 1. 제품 정의 (Product)

### 비전·노스스타
- **비전:** 가방 꾸미기의 발견→검증→득템→소유서사→자랑이 한 흐름으로 이어지는 전용 네트워크.
- **노스스타 지표(NSM):** **주간 활성 업로더 수 × 룩당 구매(샵더룩) 클릭 전환.**
  - 발견 앱이 죽는 단 하나의 이유는 *공급 고갈* → **주간 업로드율**을 단일 최우선 지표로 관리.
- **포지셔닝:** *"옷은 무신사, 가방 꾸미기는 백꾸."* (인접 패션 플랫폼에 올라타는 포지션)

### 핵심 가치 루프
사진 올림(플렉스) → 아이템 태그(발견 데이터) → 남이 "어디 거?" → 샵더룩으로 구매(커머스) → 따라 만들기 → 다시 올림.

### MoSCoW 로드맵
- **v0 (Must / 검증):** 인스타 큐레이션 계정 + 웨이트리스트 랜딩. *앱 없음.* (아래 §9 Phase 0)
- **v1 (Must / MVP):** 게스트 우선 피드 · 사진 업로드 · **사진 위 아이템 태그 핀**(오늘의집 시그니처) · 샵더룩 구매링크 · 컬렉션(입양 다이어리) · 카카오 로그인.
- **v2 (Should):** 이미지 유사검색(역검색) · A/B 투표 · 정품 검증 배지 · 셀러 입점 · 위클리 "이주의 백꾸".
- **v3 (Could):** AR 미리 얹어보기 · 자체 IP 한정 키링 드롭 · 리셀 시세 지수.

## 2. 타깃·정체성 (Anthropologist)

**핵심 통찰:** 백꾸는 다꾸·폰꾸 중 유일하게 *집 밖에서 움직이며 불특정 다수에게 노출되는* 꾸미기 = **"들고 다니는 인스타 프로필".** 그래서 사회적 신호 욕구가 가장 세고, 자랑의 대상이 *물건 → 안목(조합 센스)* 으로 이동 중이다.

**언어가 곧 정서:** 사람들은 키링을 "샀다"가 아니라 **"입양했다 / 데려왔다"**고 말한다. → 카피에서 "구매/장바구니"를 **"입양/데려오기"**로 바꾸는 것만으로 "여긴 우리를 이해하네"를 만든다.

### 페르소나 3
| | 서아 (17·고2) | 지윤 (26·취준) | 다은 (32·직장인) |
|---|---|---|---|
| 동기 | 소속(친구와 같은 최애) | 발견·큐레이션("나만 아는 작가") | 절제된 한 끗 + 진품 안심 |
| 구매처 | 다이소·에이블리 | 젤리캣·인디작가·카리마켓 | 명품 백참·팝마트·KREAM |
| 불편 | 돈 없음, "촌스러울까봐", 검색하면 광고만 | **인스타에서 봐도 어디 산 건지 못 찾음** | "과하면 유치/안 하면 심심" 경계, 정품 불안 |

### 미충족 욕구 톱5 → 기능
1. **발견의 단절**("봤는데 못 산다") → **샵더룩/이미지 역검색** *(킬러 기능)*
2. **안목 불안**("촌스럽지 않을까") → **내 가방에 미리 얹어보기 + A/B 투표**
3. **애착·관계**("들이는 것") → **컬렉션/입양 다이어리** (이름·들인 날·사연·풀콜 게이지)
4. **진품·안전 불안**(짝퉁 발암물질에도 구매) → **정품 검증 배지 + 신뢰 셀러 인증**
5. **득템 서사**(시크릿·오픈런) → **득템 피드 + 희소도 뱃지**

**전용 앱의 정당성:** 인스타는 '보여주기', 핀터레스트는 '영감'까지만. 백꾸 부족이 원하는 *발견→검증→득템→소유서사→자랑* 의 사슬을 통째로 잇는 건 전용 앱뿐.

## 3. UX 설계

- **IA:** 5탭 — 홈(피드)/탐색(검색·그리드)/올리기/알림/마이.
- **시그니처 인터랙션:** **사진 위 태그 핀** — 룩 사진의 키링·참 위치에 핀을 찍고 탭하면 아이템 카드(이름·구매처·시세) 슬라이드업. (오늘의집 '오늘의집 PICK' 구조)
- **게스트 우선:** 둘러보기·검색은 비로그인 허용. *저장·업로드·구매링크 클릭* 등 가치 행동에서만 로그인 게이트(카카오).
- **올리기 = 스텝 위저드:** 사진 → 가방종류 → 브랜드 → 아이템 태그(검색추가) → 무드 → 제목. 각 스텝 이탈 방지·자동저장.
- **현행 코드(backku.html) 진단·수정 반영:**
  - 사진 누락 통과 버그 → 사진 필수 검증.
  - `maximum-scale` 접근성 위반 → viewport 확대 허용.
  - 터치 타깃 44pt 미만 → 버튼·핀 최소 44×44pt.

## 6. AI 해자 (Systems-thinker)

- **해자는 분류기가 아니라 카탈로그 그래프다.** 롱테일 키링/참을 식별하는 **임베딩 RAG**(FashionCLIP/SigLIP) + 유저가 채우는 *출처·시세·정품* 데이터.
- 시간이 갈수록 "이 사진 속 키링이 뭔지·어디서 얼마에 사는지" 맞히는 정확도가 올라가는 **데이터 복리** = StockX식 가격 권위로 확장 가능.
- 1인 운영 현실: v1은 *유저 수동 태그 + 검색*, v2에서 유사검색 보조, 자동화는 데이터가 쌓인 뒤.

## 7. 수익 구조 (Analyst) — ★기획서에서 가장 중요한 재정의

**어필리에이트 단독은 사업이 아니다.** 키링 객단가가 낮아 건당 수수료 300~900원 → GMV가 커도 매출은 얇다. "$20M"은 *GMV*지 매출이 아니다. 현실적 3년 매출 = **연 10~30억 규모**(공격적 가정).

**수익 우선순위(재배열):**
1. **입점·셀러 구독/수수료** — 키링 공방·셀러에게 "쇼룸+판매채널" 제공, 노출·전환 기반 과금. (가장 두꺼운 그릇)
2. **브랜드 광고·기획전** — 팝마트·산리오·패션플랫폼 신상 노출. 트래픽이 자산.
3. **자체 IP 한정 굿즈** — 라부부식 블라인드/한정 드롭(브랜드 자산화 + 바이럴).
4. **어필리에이트** — 보조 수익 + *발견→구매 UX 완결용*(매출 주력 아님).
5. (장기) 리셀 시세 데이터·중개 제휴.

> **결론: "발견 미디어로 트래픽 → 입점·광고·자체IP로 수익화." 어필리에이트는 사업의 엔진이 아니라 윤활유.**

## 9. GTM·콜드스타트 플레이북 (Marketer) — "빌드보다 먼저"

### Phase 0→1,000 (0~8주): "창업자가 콘텐츠다"
**앱이 아니라 인스타/틱톡 큐레이션 계정부터.** 앱은 웨이트리스트로만.
- 채널 딱 2개: **인스타 릴스 + 틱톡**(핀터레스트 보조). 매일 1포스트.
- 포맷: 백꾸 비포/애프터, "○○년생 키링 조합", "라부부 짝퉁 vs 정품". **"저거 어디 거?" 질문에 답해주는 계정**으로 포지셔닝 → "그거 모아둔 앱 만들었어요" 전환.
- 오프라인 시드: 여대·미대·올영 앞 백꾸 스티커팩+QR 200장.
- KPI: 웨이트리스트 1,000 / 팔로워 3,000 / 평균 Save 100+.

### Phase 1,000→10,000 (8~24주): "공급 시딩 + 인플루언서"
- **키링 공방·마이크로 인플루언서 50팀** "백꾸 크리에이터" 권한(쇼룸+판매채널 제공 → 공급 엔진).
- 지그재그(20대 비율 최고)·에이블리 인접 협업("이 코디엔 이 백꾸").
- 핀터레스트 SEO 핀 200개 적립(장기 검색 자산).
- KPI: MAU 10,000 / 룩 2,000건 / 구매처 태그율 70% / W1 재방문 25%.

### 공급 3겹(닭-달걀 해법)
1. 창업자 시딩(빈 피드 절대 금지) → 2. 셀러/공방 시딩(올릴 동기 명확) → 3. 유저 UGC(목표 상태).
**원칙: 커머스(구매처 태그)를 먼저 붙여라.** 모든 룩에 구매링크 1개 이상.

### 바이럴 루프 & 캠페인
- 업로드 시 **워터마크 박힌 "백꾸 카드"** 자동생성 → 스토리 공유 → 앱 밖 도달.
- 캠페인: ①#나의최애키링 챌린지 ②"라부부 말고 너만의 백꾸"(품절 좌절 수요 흡수) ③자체 마스코트 키링 한정 드롭.

## 10. 레드팀 검증 — 반드시 통과해야 할 3가지 (Contrarian)

레드팀의 경고를 기획서에 *그대로* 박는다. 무시하면 죽는다.

1. **트렌드 휘발성:** 라부부/팝마트는 이미 정점 신호(시총 급락·리셀가 하락). → **특정 IP가 아니라 "백꾸 행동 자체"에 베팅**(다꾸→폰꾸→백꾸로 5년 이어진 메타트렌드). IP는 갈아끼운다.
2. **콜드스타트가 1번 리스크:** 크리에이터는 인스타를 떠날 이유가 없다. → **"우리가 인스타를 대체"가 아니라 "인스타에서 못 푸는 샵더룩을 백꾸가 푼다"**로 보완재 포지션. 창업자 큐레이션으로 공급 부트스트랩.
3. **수익 ≠ GMV:** §7대로 입점·광고·자체IP로 수익화.

### 빌드 전 검증 게이트 (레드팀 핵심 권고)
> **앱을 짜기 전에, 30일간 백꾸 큐레이션 인스타 계정을 직접 운영하라.**
> 통과 기준(예): 30일 팔로워 1,000+ · "어디 거예요?" 댓글/DM 주당 50+ · 웨이트리스트 300+.
> → 통과하면 v1 빌드. 미달이면 채널·포맷·타깃을 고쳐 재검증. *이게 가장 싸고 빠른 보험.*

## 11. 통합 실행 순서 (한 장 요약)

1. **(지금)** 인스타/틱톡 백꾸 큐레이션 계정 개설 + 웨이트리스트 랜딩 → 30일 검증 게이트.
2. **검증 통과 시** v1 빌드(Next.js+Supabase+카카오): 피드·사진위 태그핀·샵더룩·컬렉션. 법무 체크리스트 동시 적용(14세 차단·중개 고지·광고표시·신고창구).
3. 셀러 50팀 입점 시딩 → 공급·수익(입점) 동시 가동.
4. v2: 이미지 역검색·정품 배지·A/B 투표·위클리 픽.
5. v3: 자체 IP 한정 드롭·AR·리셀 시세 지수.

**한 문장:** *수요는 있고 빈칸도 있다. 이기는 길은 "얇게 검증 → 공급 먼저 → 입점·광고·자체IP로 수익화", 그리고 카피는 "구매"가 아니라 "입양"으로.*
