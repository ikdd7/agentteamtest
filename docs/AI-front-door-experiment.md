# AI Front Door — 컨셉 확정 + 1주 검증

## 1. 컨셉 확정 (세그먼트 · 포지셔닝 · 네이밍 · 가격)

### 타깃 세그먼트
- **시드(빠른 확산): 구직 개발자·디자이너.** 동기 최강(채용담당 24시간 응대) + 이미 링크를 콜드메일·LinkedIn에 뿌림. 단 취업하면 이탈(churn 내장).
- **본진(지속·수익): 프리랜서·1인 컨설턴트.** 영구히 영업 중 → 링크 노출이 영구 → 누적 K·LTV↑, 돈과 직결.
- 전략: **구직자로 불 붙이고 → 프리랜서로 굳힌다.**

### 포지셔닝
- ❌ "AI 프로필 / 아바타"(Linktree·HeyGen과 충돌, 노벨티) ❌
- ✅ **"24시간 일하는 내 영업·구직 비서"** — 아웃컴(대화 + 리드 수집)으로.
- 한 줄: **"Calendly가 미팅 잡기를 자동화했듯, 우리는 '당신에 대한 질문 응대'를 자동화한다."**

### 네이밍 후보
- **Doorman**(관문지기 — 메타포 직관, 기억 쉬움) ← 추천
- **AskMe / Asky**(무엇이든 물어보세요 — 행동 직결)
- **Frontdoor**(직설적이나 일반명사라 검색·상표 약함)

### 가격 (Calendly/Loom 모델)
- **Free**: 관문 1개 · 기본 응대 · **"Make your own" 배지 노출(=바이럴 엔진)**.
- **Pro $12/월**: 커스텀 도메인 · 자동응대 초안 · 배지 제거 · 리드 무제한 · 방문자 분석.
- 핵심: **무료가 배지를 달고 다녀야 바이럴이 돈다.** 배지 제거를 유료 혜택으로(Calendly 패턴).

## 2. 1주 검증 실험 (빌드 전/병행, 가장 싸게 K 가늠)

### 검증할 단 하나의 질문
> 사람들이 **"내 연락처를 AI 관문 하나로 강제"하는 행동**을 받아들이나? 그리고 **바이럴 루프(Make your own)가 클릭되나?**

### 셋업
1. 빌더 본인 + **시드 10명**(멋진 공개 프로필의 구직 개발자/프리랜서)이 각자 관문을 만든다.
2. 각자 관문 링크를 **실제 바이오 · 이메일 서명 · 콜드DM**에 단다(= 강제 노출).
3. `frontdoor` MVP를 작은 호스트(Render/Fly)에 올리고 페이지뷰·클릭만 로깅.

### 측정 지표 & 합격선
| 지표 | 무엇 | 합격선 |
|------|------|--------|
| 방문자 대화율 | 방문 중 AI와 1턴 이상 대화 | **≥ 40%** |
| 용건 남김률 | 방문 중 리드 남김 | **≥ 5%** (소유자 가치 증명) |
| **Make-your-own 클릭률** | 방문 중 CTA 클릭 (= **K 프록시**) | **≥ 2%** |
| 정성 | 채용담당/고객이 "AI랑 대화" 거부감 없나 | 시드 10명 중 ≥ 7명 "쓸만" |

→ 하나라도 크게 미달이면 포지셔닝/세그먼트 피벗. 통과면 본격 빌드.

### 시드 아웃리치 템플릿
**빌드인퍼블릭(X/Indie Hackers):**
> "AI 버전의 나를 만들었다. 내 이력을 학습해서, 채용담당·고객이 나한테 24시간 질문할 수 있고 용건은 내 인박스로 쌓인다. 링크 하나가 내 연락처 전부. 베타 10명 무료 → [link]"

**구직 커뮤니티(r/cscareerquestions·디스코드) DM:**
> "콜드메일 회신율 낮아서 만든 건데 — 이력서 링크 대신 '나한테 뭐든 물어보는 AI 링크'를 넣었더니 반응이 달라요. 무료로 만들어 드릴게요, 피드백만."

**프리랜서(r/forhire·크몽 인접):**
> "잠재고객 문의에 매번 같은 답 쓰지 마세요. 당신을 학습한 AI가 24시간 응대하고, 진짜 용건만 인박스로. 링크 하나로 영업."

## 3. 지금 만든 것 (`frontdoor/`)
MVP 코어가 **키 없이(mock LLM)도 동작**: 페르소나 적재 → 방문자 질문 근거 응대 + attribution → 용건이 소유자 인박스로 → 일일 요약. 방문자 관문 페이지(공유 링크 목업)도 생성.
```bash
python -m frontdoor                 # mock 데모(키 0)
python -m frontdoor --html door.html  # 방문자 관문 페이지 저장
python -m frontdoor --llm anthropic --profile me.txt   # 실제 Claude(본인 PC)
```
실서비스는 키워드 검색 → 임베딩 RAG, CLI → Next.js 프런트로 확장.
