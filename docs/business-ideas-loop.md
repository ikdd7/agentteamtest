# 욕구이론 × 특정 상황·니즈 사업 아이디어 (자동 누적)

> 30분마다 10개씩 자동 생성. 각 아이디어는 **〈욕구이론의 구체 욕구 → 또렷한 타깃 상황 → 해결 메커니즘〉** 구조로 정리한다.
> 생성 방식: Claude Code `/loop` + 30분 간격 Monitor 트리거. 채팅에 출력하고 이 파일에 누적 append.

---

## 1차 배치 — 2026-06-23

**1. 거지맵 2.0 — "공짜밥 레이더"**
- 욕구: 생리적 욕구(음식) + 안전 욕구(경제적 안정)
- 상황: 월말 잔고 0원인 대학생·사회초년생
- 아이디어: 무료시식·1+1·교회/관공서 무료급식·앱 첫주문 쿠폰을 위치+시간순으로 모아주는 지도. "오늘 3km 안에서 0원으로 먹을 수 있는 끼니" 알림.

**2. 대다모(탈모) — "두피 타임랩스"**
- 욕구: 존중 욕구(외모·자존감) + 인지적 욕구(원인 파악)
- 상황: 탈모 초기, 병원 가기는 민망한 2030 남성
- 아이디어: 매주 같은 각도로 정수리 사진을 찍으면 AI가 밀도 변화를 그래프화. 약·샴푸 효과를 객관 수치로 보여주고 익명 또래 비교.

**3. 차바조(차박+바베큐) — "오늘 어디서 자지"**
- 욕구: 안전 욕구(안심하고 잘 곳) + 자율성(구속 없는 여행)
- 상황: 차박러가 밤늦게 합법·안전한 박지를 못 찾는 상황
- 아이디어: 화장실·소음·CCTV·평탄도·금지구역 여부를 실시간 크라우드 평점으로 제공하는 차박 전용 지도 + 옆자리 차박러 매칭.

**4. 혼틈 — "혼자인 거 티 안 나게"**
- 욕구: 소속 욕구 + 굴욕회피(머레이)
- 상황: 혼밥·혼영·혼술이 부담스러운 사람
- 아이디어: 같은 시간·같은 식당에 "말은 안 걸고 따로 앉되 혼자는 아닌" 느슨한 동석 매칭. 대화 의무 0%.

**5. 돈쭐내장 — "착한 가게 원정대"**
- 욕구: 자기초월 욕구(이타·의미) + 과시 욕구
- 상황: 선행 가게를 "돈쭐"내고 싶지만 어디인지 모르는 소비자
- 아이디어: 선행/미담 가게를 모아 단체 방문 이벤트를 열고, 방문 인증 배지로 "착한 소비 랭킹". 가게엔 매출, 소비자엔 의미+인정.

**6. 갱생모드 — "작심삼일 강제소환"**
- 욕구: 자기실현 욕구 + 유능감(자기결정이론)
- 상황: 매번 운동·공부 다짐 후 3일 만에 포기하는 사람
- 아이디어: 목표 실패 시 미리 맡긴 보증금이 "내가 싫어하는 단체"에 기부됨(반작용 욕구 자극). 성공 시 또래 그룹이 실시간 응원.

**7. 인맥세탁소 — "관계 정리 대행"**
- 욕구: 관계성 + 거부 욕구(머레이: 불필요한 관계 배제)
- 상황: 연락처·카톡·SNS에 부담스러운 관계가 쌓인 사람
- 아이디어: 1년+ 무연락·일방향 관계를 진단해 "정리/유지/복원" 추천. 어색하지 않게 안부 멘트도 자동 제안.

**8. 효도자판기 — "부모님 디지털 대리효도"**
- 욕구: 양육 욕구(역방향) + 죄책감 해소
- 상황: 멀리 사는 부모님께 신경 못 쓰는 직장인 자녀
- 아이디어: 매주 부모님 댁에 반찬·약·생필품을 자동 배송하고, 기사님이 안부 영상 1분을 찍어 자녀에게 전달. "효도 구독".

**9. 텅장알리미 — "월급 도착 → 자동 생존세팅"**
- 욕구: 안전 욕구(질서·통제) + 위험회피
- 상황: 월급 들어오면 며칠 만에 다 쓰는 사람
- 아이디어: 월급 입금 즉시 고정비·비상금·"이번 달 생존 가능 일수"를 자동 분리/표시. 과소비 직전엔 "지금 사면 D-3 굶음" 경고.

**10. 눈치제로 — "퇴근각 알림봇"**
- 욕구: 자율성 + 굴욕회피(눈치 스트레스)
- 상황: 상사 눈치 보느라 정시 퇴근 못 하는 직장인
- 아이디어: 팀 분위기·상사 일정·동료 퇴근 데이터를 익명 집계해 "오늘은 6:05이 안전 퇴근각" 신호 제공. 사내 정시퇴근 문화 데이터화.
