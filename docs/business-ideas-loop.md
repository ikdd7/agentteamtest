# 욕구이론 × 특정 상황·니즈 사업 아이디어 (자동 누적)

> 30분마다 10개씩 자동 생성. 각 아이디어는 **〈욕구이론의 구체 욕구 → 또렷한 타깃 상황 → 해결 메커니즘〉** 구조로 정리한다.
> 생성 방식: Claude Code `/loop` + 30분 간격 Monitor 트리거. 채팅에 출력하고 이 파일에 누적 append.

---

## 1차 배치 — 2026-06-23

**1. 거지맵 2.0 — "공짜밥 레이더"**
- 욕구: 생리적 욕구(음식) + 안전 욕구(경제적 안정)
- 상황: 월말 잔고 0원인 대학생·사회초년생
- 아이디어: 무료시식·1+1·교회/관공서 무료급식·앱 첫주문 쿠폰을 위치+시간순으로 모아주는 지도. "오늘 3km 안에서 0원으로 먹을 수 있는 끼니" 알림.

**2. 대다모(탈모) — "두피 타임랩스"**
- 욕구: 존중 욕구(외모·자존감) + 인지적 욕구(원인 파악)
- 상황: 탈모 초기, 병원 가기는 민망한 2030 남성
- 아이디어: 매주 같은 각도로 정수리 사진을 찍으면 AI가 밀도 변화를 그래프화. 약·샴푸 효과를 객관 수치로 보여주고 익명 또래 비교.

**3. 차바조(차박+바베큐) — "오늘 어디서 자지"**
- 욕구: 안전 욕구(안심하고 잘 곳) + 자율성(구속 없는 여행)
- 상황: 차박러가 밤늦게 합법·안전한 박지를 못 찾는 상황
- 아이디어: 화장실·소음·CCTV·평탄도·금지구역 여부를 실시간 크라우드 평점으로 제공하는 차박 전용 지도 + 옆자리 차박러 매칭.

**4. 혼틈 — "혼자인 거 티 안 나게"**
- 욕구: 소속 욕구 + 굴욕회피(머레이)
- 상황: 혼밥·혼영·혼술이 부담스러운 사람
- 아이디어: 같은 시간·같은 식당에 "말은 안 걸고 따로 앉되 혼자는 아닌" 느슨한 동석 매칭. 대화 의무 0%.

**5. 돈쭐내장 — "착한 가게 원정대"**
- 욕구: 자기초월 욕구(이타·의미) + 과시 욕구
- 상황: 선행 가게를 "돈쭐"내고 싶지만 어디인지 모르는 소비자
- 아이디어: 선행/미담 가게를 모아 단체 방문 이벤트를 열고, 방문 인증 배지로 "착한 소비 랭킹". 가게엔 매출, 소비자엔 의미+인정.

**6. 갱생모드 — "작심삼일 강제소환"**
- 욕구: 자기실현 욕구 + 유능감(자기결정이론)
- 상황: 매번 운동·공부 다짐 후 3일 만에 포기하는 사람
- 아이디어: 목표 실패 시 미리 맡긴 보증금이 "내가 싫어하는 단체"에 기부됨(반작용 욕구 자극). 성공 시 또래 그룹이 실시간 응원.

**7. 인맥세탁소 — "관계 정리 대행"**
- 욕구: 관계성 + 거부 욕구(머레이: 불필요한 관계 배제)
- 상황: 연락처·카톡·SNS에 부담스러운 관계가 쌓인 사람
- 아이디어: 1년+ 무연락·일방향 관계를 진단해 "정리/유지/복원" 추천. 어색하지 않게 안부 멘트도 자동 제안.

**8. 효도자판기 — "부모님 디지털 대리효도"**
- 욕구: 양육 욕구(역방향) + 죄책감 해소
- 상황: 멀리 사는 부모님께 신경 못 쓰는 직장인 자녀
- 아이디어: 매주 부모님 댁에 반찬·약·생필품을 자동 배송하고, 기사님이 안부 영상 1분을 찍어 자녀에게 전달. "효도 구독".

**9. 텅장알리미 — "월급 도착 → 자동 생존세팅"**
- 욕구: 안전 욕구(질서·통제) + 위험회피
- 상황: 월급 들어오면 며칠 만에 다 쓰는 사람
- 아이디어: 월급 입금 즉시 고정비·비상금·"이번 달 생존 가능 일수"를 자동 분리/표시. 과소비 직전엔 "지금 사면 D-3 굶음" 경고.

**10. 눈치제로 — "퇴근각 알림봇"**
- 욕구: 자율성 + 굴욕회피(눈치 스트레스)
- 상황: 상사 눈치 보느라 정시 퇴근 못 하는 직장인
- 아이디어: 팀 분위기·상사 일정·동료 퇴근 데이터를 익명 집계해 "오늘은 6:05이 안전 퇴근각" 신호 제공. 사내 정시퇴근 문화 데이터화.


## 2차 배치 — 2026-06-23

**1. 귀가시그널 — "나 집 잘 들어감 자동중계"**
- 욕구: 안전 욕구(신체적 안전) + 구원 욕구(도움받음)
- 상황: 늦은 밤 혼자 귀가하는 1인가구 여성·청년
- 아이디어: 귀가 경로를 지정하면 도착까지 보호자에게 실시간 위치가 중계되고, 경로 이탈·정지 5분 시 자동 알림. 도착 시 "집 도착" 한 줄만 전송돼 매번 보고하는 피로를 없앤다.

**2. 약충돌봇 — "이 약 같이 먹어도 돼?"**
- 욕구: 인지적 욕구(이해) + 위험회피 욕구
- 상황: 여러 병원 약·영양제를 동시에 먹는 중장년·만성질환자
- 아이디어: 복용 중인 약·영양제를 찍으면 성분 상호작용·중복을 경고하고 "이 조합은 위장 부담" 식으로 알려준다. 복용 시간표와 리필 알림까지 묶어 관리.

**3. 사이드버디 — "회사 밖 가짜동료"**
- 욕구: 소속 욕구 + 관계성(자기결정이론)
- 상황: 재택·프리랜서로 동료 없이 일하는 사람
- 아이디어: 같은 시간대에 각자 일하는 사람끼리 화면 없이 "작업 중" 상태만 공유하는 느슨한 동석 룸. 시작·점심·마감 때만 가볍게 인사해 고립감만 덜어준다.

**4. 명절알리바이 — "잔소리 방어 시나리오"**
- 욕구: 자율성 욕구 + 굴욕회피 욕구
- 상황: 명절마다 결혼·취업·연봉 잔소리에 시달리는 2030
- 아이디어: 친척 질문 유형별 받아치기 멘트와 화제 전환 카드를 미리 제공하고, 적당한 시점에 울릴 "긴급 전화" 알람을 예약. 탈출 명분을 설계해준다.

**5. 펫별나라 — "무지개다리 추모관"**
- 욕구: 양육 욕구 + 자기초월 욕구(의미·애도)
- 상황: 반려동물을 떠나보낸 펫로스 증후군 보호자
- 아이디어: 사진·영상으로 추모 페이지를 만들고, 기일·생일에 회상 알림을 보낸다. 같은 상실을 겪은 사람들의 익명 위로 편지와 추모 모임을 연결.

**6. 무지출인증 — "0원으로 며칠 버티나"**
- 욕구: 존중 욕구(성취·인정) + 안전 욕구(절약)
- 상황: 짠테크·무지출 챌린지를 혼자 하다 흐지부지되는 사람
- 아이디어: 카드·페이 내역을 읽어 "무지출 연속일"을 자동 집계하고 배지로 인증. 같은 챌린지 그룹에서 랭킹·응원으로 동기를 유지하고, 절약분을 자동 저축으로 전환.

**7. 길치내비 — "랜드마크로 알려주는 길안내"**
- 욕구: 유능감 욕구 + 굴욕회피 욕구
- 상황: 방향·지도를 잘 못 읽는 심한 길치
- 아이디어: "북쪽 200m" 대신 "파란 간판 카페에서 왼쪽, 편의점 보이면 직진"처럼 눈에 보이는 랜드마크로 안내. 골목·실내(지하상가·역사)에서도 사진 기반으로 길을 잡아준다.

**8. 짠손손질 — "못 버리는 사람 대신 결정"**
- 욕구: 질서 욕구(정돈) + 위험회피(후회 회피)
- 상황: 물건을 못 버려 집이 점점 좁아지는 사람
- 아이디어: 물건을 찍으면 "최근 1년 사용 여부"를 물어 보관/기부/판매/폐기를 추천하고, 중고 시세와 기부처를 바로 연결. "버리면 후회" 불안을 90일 보관함으로 완충.

**9. 작아진옷장 — "안 입는 옷 자동 순환"**
- 욕구: 심미 욕구 + 자기초월(나눔)
- 상황: 옷은 많은데 늘 입을 게 없는 사람
- 아이디어: 옷장을 등록하면 착용 빈도를 추적해 "1년째 안 입은 옷"을 골라 교환·기부·판매로 순환. 남은 옷으로 오늘 날씨에 맞는 코디를 추천해 새로 사지 않게 한다.

**10. 빚갚기레이스 — "대출 박멸 게임"**
- 욕구: 성취 욕구(nAch) + 반작용 욕구(만회)
- 상황: 학자금·카드론에 눌려 갚을 엄두가 안 나는 사회초년생
- 아이디어: 여러 빚을 한곳에 모아 "눈덩이/고금리 우선" 상환 시나리오를 게임화. 갚을 때마다 줄어드는 게이지와 완납 예상일을 보여주고, 같은 목표의 익명 동료들과 진척을 겨룬다.


## 3차 배치 — 2026-06-23

**1. 퇴사각보고서 — "그만둬도 되나 진단"**
- 욕구: 자기실현 욕구 + 안전 욕구(경제적 안정)
- 상황: 퇴사·이직을 고민하지만 결단 못 내리는 직장인
- 아이디어: 현재 저축·생활비·예상 공백기를 입력하면 "버틸 수 있는 개월 수"와 사이드 수입 시나리오를 계산. 충동 퇴사 전 "지금 그만두면 D-OOO 생존" 현실 지표를 보여준다.

**2. 다시일해 — "경단녀 복귀 워밍업"**
- 욕구: 존중 욕구(자존감·인정) + 유능감
- 상황: 출산·육아로 경력이 끊겼다 복귀하려는 사람
- 아이디어: 공백기를 약점이 아닌 서사로 바꾸는 이력서·면접 코칭과, 짧게 감 잡는 단기 프로젝트를 연결. 같은 처지 동료들과 복귀 스터디로 자신감을 회복.

**3. 꿀잠루틴 — "잠 못 드는 밤 처방"**
- 욕구: 생리적 욕구(수면) + 감각 욕구(이완)
- 상황: 자려고 누우면 생각이 많아 못 자는 사람
- 아이디어: 취침 전 행동·카페인·화면 사용을 기록해 불면 패턴을 찾아주고, 그날 상태에 맞는 호흡·소리·루틴을 처방. 수면 일기로 무엇이 효과 있었는지 객관화.

**4. 동네친구 — "반경 1km 취향메이트"**
- 욕구: 친애 욕구(머레이) + 소속 욕구
- 상황: 이사·자취 후 동네에 아는 사람이 없는 사람
- 아이디어: 같은 동네의 비슷한 취향(러닝·전시·맛집)인 사람을 소규모로 매칭해 가벼운 동네 모임을 연다. 멀리 안 가도 되는 "걸어서 만나는 관계"에 집중.

**5. 거절대행소 — "싫다고 대신 말해줌"**
- 욕구: 자율성 욕구 + 굴욕회피 욕구
- 상황: 부탁·영업·모임을 거절 못 해 끌려다니는 사람
- 아이디어: 상황별로 상대 기분 안 상하게 거절하는 멘트를 만들어주고, 필요하면 대신 통보해주는 메시지 템플릿을 제공. "착한 사람" 부담에서 벗어나게 한다.

**6. 약관세줄 — "동의 누르기 전 핵심만"**
- 욕구: 인지적 욕구(이해) + 위험회피 욕구
- 상황: 계약서·약관·동의서를 안 읽고 그냥 누르는 사람
- 아이디어: 약관을 넣으면 "독소조항·자동결제·해지 위약금"만 세 줄로 요약하고 위험도를 신호등으로 표시. 서명 전에 "여기 조심" 포인트를 짚어준다.

**7. 전세안심렌즈 — "이 집 사기 위험도"**
- 욕구: 안전 욕구(주거) + 위험회피 욕구
- 상황: 전세·월세 계약을 앞둔, 전세사기가 무서운 청년
- 아이디어: 등기부·시세·보증금 비율을 분석해 깡통전세·이중계약 위험을 점수화하고, 확인할 서류와 특약 문구를 안내. 계약 직전 마지막 체크리스트로 보호.

**8. 처음육아 — "오늘 뭐 해줘야 하지"**
- 욕구: 양육 욕구 + 인지적 욕구(이해)
- 상황: 정보 홍수 속에 헤매는 초보 부모
- 아이디어: 아이 개월 수에 맞춰 이유식·예방접종·발달 체크를 "오늘 할 일"로 한 장에 정리. 검증된 정보만 큐레이션하고, 같은 월령 부모 Q&A로 불안을 덜어준다.

**9. 주말뭐하지 — "즉흥 놀거리 뽑기"**
- 욕구: 유희 욕구 + 자율성 욕구
- 상황: 주말마다 "뭐하지" 하다 그냥 누워서 끝내는 사람
- 아이디어: 날씨·예산·기분·동행 여부를 고르면 근처에서 지금 할 수 있는 놀거리를 즉석 추천. 결정 장애를 "랜덤 뽑기"로 깨고, 가본 곳은 기록해 다음 추천에 반영.

**10. 엔딩노트 — "미리 남기는 마지막 정리"**
- 욕구: 자기초월 욕구(의미·유산) + 질서 욕구
- 상황: 갑작스러운 일에 대비해 신변을 정리해두고 싶은 사람
- 아이디어: 계좌·구독·디지털 계정·장례 희망·남길 메시지를 한곳에 비공개로 기록하고, 지정한 사람에게 특정 조건에서만 공개. 남은 가족의 막막함을 줄여준다.


## 4차 배치 — 2026-06-23

**1. 운동인증각 — "PT보다 무서운 또래 감시"**
- 욕구: 존중 욕구(과시·인정) + 유능감
- 상황: 헬스 등록해놓고 3일 만에 안 가는 사람
- 아이디어: 운동 인증샷을 소규모 그룹에 올려 출석을 공유하고, 빠지면 벌금이 적립돼 성실한 멤버에게 분배. 트레이너 없이 또래 압박으로 꾸준함을 만든다.

**2. 방꾸요정 — "있는 가구로 방 살리기"**
- 욕구: 심미 욕구 + 자율성 욕구
- 상황: 칙칙한 자취방을 돈 안 들이고 바꾸고 싶은 사람
- 아이디어: 방 사진을 올리면 기존 가구 재배치안과 5만 원 이하 소품 추천을 비주얼로 제안. 클릭 한 번에 구매 링크까지 연결해 "현실 가능한 변신"만 보여준다.

**3. 멀리커플 — "장거리 연애 루틴메이커"**
- 욕구: 친애 욕구 + 관계성(자기결정이론)
- 상황: 시차·거리로 소원해지기 쉬운 장거리 커플
- 아이디어: 같이 보는 영화 동시 재생, 함께 하는 미션, 기념일 카운트다운을 한 앱에서. 연락이 뜸해지면 "오늘 한 마디" 가벼운 넛지로 관계 온도를 유지.

**4. 부모님안녕 — "멀리 사는 부모 건강 신호등"**
- 욕구: 양육 욕구(역방향) + 안전 욕구(건강)
- 상황: 따로 사는 부모님 건강이 늘 걱정인 자녀
- 아이디어: 부모님 폰·웨어러블의 활동량·복약·통화 빈도로 일상 이상 신호를 감지해 자녀에게 부드럽게 알림. 매일 캐묻지 않아도 "오늘도 평소대로"를 확인.

**5. 주식말사전 — "차트 용어 한국어로"**
- 욕구: 인지적 욕구(이해) + 위험회피 욕구
- 상황: 투자 시작했지만 용어·공시가 외계어인 초보
- 아이디어: 모르는 용어·공시·리포트를 넣으면 쉬운 말로 풀고 "그래서 내 돈엔 무슨 의미"까지 해석. 충동매매 전 핵심 위험을 짚어주는 가드레일.

**6. 합격페이스 — "혼자 공부 페이스메이커"**
- 욕구: 성취 욕구(nAch) + 질서 욕구
- 상황: 자격증·고시를 독학하다 계획이 무너지는 사람
- 아이디어: 시험일에서 역산한 진도표를 자동 생성하고, 매일 분량과 모의고사 시점을 관리. 같은 시험 준비생들과 진척률을 비교해 페이스를 유지.

**7. 발표리허설 — "떨리는 발표 미리 연습"**
- 욕구: 유능감 욕구 + 굴욕회피 욕구
- 상황: 사람 앞 발표·면접만 하면 머리가 하얘지는 사람
- 아이디어: 대본을 넣고 말하면 속도·군말("어…")·시선 분량을 분석해 피드백. 예상 질문과 답변 연습으로 실전 전 무대 공포를 줄인다.

**8. 재능한스푼 — "남는 재능 나눠주기"**
- 욕구: 자기초월 욕구(이타) + 존중 욕구(인정)
- 상황: 가진 기술을 의미 있게 쓰고 싶은 직장인·은퇴자
- 아이디어: 디자인·번역·코딩·악기 같은 재능을 비영리·소상공인 도움 요청과 매칭. 짧게 기여하고 감사 후기를 받아 "쓸모 있는 사람"이라는 효능감을 채운다.

**9. 산책메이트 — "강아지 산책 같이 갈래?"**
- 욕구: 양육 욕구 + 소속 욕구
- 상황: 바빠서 반려견 산책을 충분히 못 시키는 보호자
- 아이디어: 동네 견주끼리 산책 시간을 맞춰 함께 걷거나, 검증된 산책 도우미와 연결. 강아지도 사람도 외롭지 않은 "같이 산책"을 만든다.

**10. 구독다이어트 — "줄줄 새는 정기결제 색출"**
- 욕구: 안전 욕구(통제) + 질서 욕구
- 상황: 안 쓰는 구독에 매달 돈이 빠져나가는 사람
- 아이디어: 결제 내역에서 정기결제를 모아 "최근 사용 여부"와 함께 보여주고, 해지를 한 번에 도와준다. 무료체험 종료 전 알림으로 자동결제 함정을 차단.
