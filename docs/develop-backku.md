# 딥 디벨롭 — 백꾸 도감 (가방 꾸미기 × 도감 × 출처·시세)

> 누적 톱 후보(80점)를 호구지수처럼 끝까지 깎는다. 경쟁·시장 WebSearch 실측 반영.
> 한 줄: **내 백꾸(키링·인형·스트랩)를 자랑하고, 아이템 *출처·시세*를 태깅하고, 도감으로 수집하는 네트워크.**

## 0. 왜 이 칸인가 (실측 근거)
- **트렌드 폭발:** 키링 거래액 +178%, 인형키링 +28%, 스티커 +219%(별다꾸 확산). 라부부 KREAM 거래액 *전년比 +7,711%*, 리셀 34배 프리미엄.
- **전용 앱 부재:** 백꾸/키링 *도감·커뮤니티* 전용 앱 없음. 거래는 KREAM·번개·G마켓·아이디어스, 자랑은 인스타·틱톡으로 *흩어짐*.
- **"어디서 샀어?" 욕구 실재:** "백꿈 오리인형 구매처 찾았어요"가 틱톡 검색어. 발견 마찰이 큼.
→ *거대 트렌드 × 전용 소셜앱 없음*의 교과서. 단, 거래(KREAM/번개)는 인접 → 우리는 *꾸미기 큐레이션+도감+출처+시세*로 차별.

## 1. 욕구 3종 (왜 올리고/보나)
- ① **플렉스(자랑):** "내 가방 이렇게 꾸몄다" — OOTD式 자기표현. 공급 동기.
- ② **발견("어디서 샀어?"):** 남의 백꾸 보고 *그 아이템 출처*를 알고 싶음. 수요 동기 + 검색유입.
- ③ **수집·시세:** 한정판 키링(라부부·스마이스키·쁘넥도) 도감 완성욕 + 리셀 프리미엄 추적. 락인 + 데이터.

## 2. 발칙한 form (3층)
- **훅(바이럴) = "내 백꾸 시가총액/점수":** 가방 사진 올리면 단 아이템들을 인식·태깅 → *총 시가(리셀 기준) + 백꾸 점수* 카드. 호구지수 메커닉 재활용. "내 가방에 30만원 매달고 다녔네ㅋㅋ" 공유.
- **앵커(리텐션) = 아이템 도감:** 인기 키링/인형을 카탈로그화 → 내가 가진 것 체크 + 위시 + "이 동네 어디서 파나". 수집 완성욕.
- **데이터 해자 = 출처·실거래 시세:** 유저가 "이거 ○○에서 ○원에 샀다"를 (자랑·발견 위해) 입력 → *출처·시세 DB*가 쌓임 = 라부부판 시세지수.

## 3. 바이럴 훅 디테일
- 가방 사진 → AI가 아이템 박스 인식 → 각 아이템 "이거 뭐예요?" 태깅(유저·AI 협업) → 시가·출처 자동 표시.
- 결과 카드: `내 백꾸 시가 28만원 💸 · 백꾸지수 87 · 가장 비싼 건 라부부 시크릿`. 하단 `내 백꾸 점수 보기 →`(루프).
- "이 키링 어디서 샀어?" 한 탭 → 출처 링크/매장(커머스 연결).

## 4. 콜드스타트 (1인 시딩)
- 창업자가 인기 아이템(라부부·스마이스키·쁘넥도·산리오·디즈니·핸드메이드 등) **도감 200~300개**를 공개정보로 시딩(이름·이미지·정가·대표 출처).
- 시세는 KREAM/번개 공개 시세로 부트스트랩 → 유저 실거래 입력으로 보정.
- 자랑 욕구가 공급을 굴려 — give-to-get 아님(올리고 싶어서 올림). 데이터 0에서도 도감·시세 작동.

## 5. 수익 ($20M 경로)
- **1차: 커머스 어필리에이트** — "어디서 샀어?" → 구매처 링크(스마트스토어·아이디어스·G마켓·팝마트) 수수료. 발견 욕구가 곧 구매전환.
- **2차: 리셀 시세·거래 연계** — 라부부式 한정판 리셀 시세 권위 → 거래 중개 수수료 or KREAM/번개 제휴. (KREAM 라부부 +7,711%가 시장 크기 증거)
- **3차: 브랜드·캐릭터 협찬** — 팝마트·산리오·캐릭터 IP·패션플랫폼(지그재그·에이블리) 신상 노출 광고.
- 그릇: 키링·굿즈·리셀 시장 + 글로벌(라부부·팝마트는 글로벌 현상) 확장 시 큼.

## 6. 네이밍
- **"백꾸"**(직관·검색강함) / **"참chip"·"키링도감"·"꾸미"** / 영문 "Charms". → 추천: 바이럴 = *백꾸*, 글로벌 확장 대비 *Charms* 병행.

## 7. MVP (1인 바이브코딩)
사진 업로드 → 아이템 태깅(도감 연결) → 시가·출처 표시 → 백꾸 점수 카드(공유) → 도감 체크리스트(가진 것/위시). 커머스는 어필리에이트 링크로 위임(ops 0). 호구지수 코드(칩·카드·공유) 상당 재활용 가능.

## 8. 경쟁 (실측)
- 전용 도감/커뮤니티: **없음(빈칸).**
- 거래: KREAM(리셀)·번개·G마켓·아이디어스 — *인접*. 차별 = 우리는 거래소가 아니라 *꾸미기 큐레이션+도감+출처발견+시세 미디어*. (KREAM이 "꾸미기 자랑·도감"을 안 함.)
- 자랑: 인스타·틱톡 — *구조화·검색·출처태깅 없음*.

## 9. 약점 (정직)
- **트렌드 휘발성:** 라부부 등 특정 IP는 식을 수 있음. → 완화: *백꾸/꾸미기 행동 자체*는 지속(다꾸→신꾸→백꾸로 5년 이어짐), IP는 갈아끼움.
- **사진 인식·정규화:** 아이템 자동 태깅 난도(초기엔 유저 수동 태깅+AI 보조).
- **거래 인큐번트:** KREAM/번개가 "꾸미기 큐레이션"을 붙이면 위협 → 선점·커뮤니티 결속으로 방어.
- **그릇:** 어필리에이트 단독은 얇음 → 리셀 시세 권위+글로벌이 그릇을 키워야.

## 10. 점수 (100)
관심강도 22 · 밈/바이럴 22(시가총액 카드+"어디서 샀어") · 경쟁공백 18(도감 전용 빔) · 그릇 14(커머스+리셀+글로벌) · 1인빌드 9 = **85** (앱-아이디어 루프 기준 80→딥디벨롭 후 85).

**판정:** *거대 트렌드 × 전용앱 없음 × 자랑+발견+수집 욕구 + 라부부 리셀이라는 깊은 지갑* — 강한 후보. 다음: 백꾸 점수 산식 + 도감 데이터 구조 + MVP 화면 플로우.
